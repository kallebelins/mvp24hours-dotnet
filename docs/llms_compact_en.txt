## Mvp24Hours Code Conventions

- Entities: use IEntityBase<TKey>, IEntityLog
- Repository: use IRepository<TEntity>, IRepositoryAsync<TEntity>
- Services: use IBusinessResult<T> for returns
- DI: use extension methods (AddMvp24HoursDbContext, AddMvp24HoursRepository)
