## Convenções de Código Mvp24Hours

- Entidades: usar IEntityBase<TKey>, IEntityLog
- Repositório: usar IRepository<TEntity>, IRepositoryAsync<TEntity>
- Serviços: usar IBusinessResult<T> para retornos
- DI: usar extension methods (AddMvp24HoursDbContext, AddMvp24HoursRepository)
